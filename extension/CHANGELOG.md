# SnapStream Changelog

## [Initial Version] - {PR_MERGE_DATE}

- Search Channels command — browse and search live TV channels grouped by category
- Import Channels command — import channels from .m3u playlist files with preview and selection
- Auto Picture-in-Picture — streams automatically float over your workspace via QuickTime
- Favorites — pin channels with Cmd+F, persisted across sessions
- Frecency sorting — most-used channels rise to the top
- Configurable remote playlist URL, auto-PiP toggle, PiP delay
